## 📖 Sobre o Projeto

Este portfólio foi desenvolvido para centralizar minhas informações profissionais, projetos e habilidades técnicas em uma única plataforma.

Além de servir como cartão de visitas digital, o projeto demonstra meus conhecimentos em desenvolvimento web por meio de uma interface moderna, responsiva e intuitiva.

## 💼 Projetos Apresentados

### 🌐 Portfólio Profissional

Site desenvolvido para apresentar minha trajetória profissional, habilidades técnicas e projetos.

### 🏦 Sistema Bancário em Python

Aplicação desktop desenvolvida utilizando Python e interface gráfica.

**Principais funcionalidades:**

* Cadastro de clientes
* Depósitos
* Saques
* Transferências
* Histórico de movimentações

### 🎉 Sistema RSVP para Convite de Aniversário

Aplicação web para gerenciamento de confirmações de presença em eventos.

**Principais funcionalidades:**

* Confirmação de presença online
* Interface amigável
* Design responsivo
* Organização de convidados
